# SKILLS.md - Habilidades Requeridas para el Proyecto NOA

Este documento detalla las habilidades técnicas y de dominio necesarias para desarrollar, mantener y escalar el asistente NOA.

## Habilidades Técnicas

### 1. Desarrollo Backend (Python)
- **Dominio de Python 3.11+**: Manejo de asincronía (`asyncio`), tipado estático (`typing`).
- **FastAPI**: Creación de APIs REST, manejo de Webhooks, dependencias y seguridad.
- **Uvicorn**: Configuración de servidor ASGI.

### 2. Inteligencia Artificial y NLP
- **OpenAI API**: Implementación de modelos GPT-4 para detección de intención y generación de respuestas.
- **Embeddings**: Uso de modelos de embeddings para representación semántica.
- **Análisis de Sentimiento**: Clasificación de sentimiento y detección de riesgos en mensajes de texto.
- **Scoring de Leads**: Implementación de algoritmos de calificación basados en múltiples factores.

### 3. Recuperación Aumentada por Generación (RAG)
- **Bases de Datos Vectoriales**: Manejo de FAISS o ChromaDB.
- **Orquestación**: Inyección de contexto, búsqueda por similitud y filtrado por metadatos.

### 4. Integraciones y Mensajería
- **Evolution API**: Gestión de sesiones de WhatsApp, envío y recepción de mensajes, manejo de estados.
- **Webhooks**: Consumo y procesamiento de eventos en tiempo real.

### 5. Gestión de Datos
- **PostgreSQL**: Diseño de esquemas para persistencia de leads y sesiones.
- **SQLite**: Gestión de bases de datos locales para entornos de desarrollo.
- **Redis**: Implementación de caché de sesiones y persistencia de corto plazo.

### 6. Infraestructura y DevOps
- **Docker & Docker Compose**: Contenerización de servicios y orquestación local.
- **Linux/VPS**: Administración básica de servidores para despliegue.
- **Nginx**: Configuración de proxies inversos.

## Habilidades de Dominio
- **Arquitectura de Software**: Diseño de Sistemas de Estados (FSM) para flujos conversacionales complejos.
- **Seguridad**: Manejo seguro de tokens, encriptación de datos y cumplimiento de normativas de privacidad.
- **Comunicación Profesional**: Capacidad para traducir requerimientos de negocio en prompts de sistema efectivos y flujos lógicos.
